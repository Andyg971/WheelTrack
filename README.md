## 🎯 Vision

WheelTrack révolutionne la gestion automobile en offrant une expérience utilisateur exceptionnelle alliant simplicité et puissance. Que vous soyez propriétaire d'un véhicule personnel, gestionnaire d'une flotte ou entrepreneur dans la location automobile, WheelTrack s'adapte à vos besoins.

## ✨ Fonctionnalités Principales

### 🏠 **Dashboard Intelligent**
- **Aperçu en temps réel** de votre parc automobile
- **Analytics avancées** avec graphiques interactifs
- **Widgets personnalisables** pour un accès rapide aux informations critiques
- **Notifications intelligentes** pour la maintenance et les échéances

### 🚘 **Gestion de Véhicules**
- **Profils détaillés** : marque, modèle, année, kilométrage, type de carburant
- **Historique complet** d'achat, maintenance et revente
- **Évaluation en temps réel** de la valeur marchande
- **Gestion complète des photos** : photo principale, galerie supplémentaire, documents numérisés
- **Compression intelligente** des images avec optimisation automatique
- **Placeholders dynamiques** basés sur la couleur et le type de véhicule
- **Statuts dynamiques** : actif, en maintenance, en location, vendu

### 💰 **Suivi des Dépenses**
- **Catégorisation automatique** : carburant, maintenance, assurance, taxes
- **Scan de reçus** avec reconnaissance OCR
- **Rapports détaillés** par période et par véhicule
- **Alertes budgétaires** personnalisables
- **Export en PDF/Excel** pour la comptabilité

### 🔧 **Maintenance Prédictive**
- **Planification intelligente** basée sur le kilométrage et le temps
- **Rappels automatiques** pour les révisions et contrôles techniques
- **Historique complet** des interventions
- **Intégration avec des garages partenaires**
- **Prédictions de coûts** basées sur l'historique

### 🏢 **Gestion de Garage**
- **Répertoire complet** de vos garages de confiance
- **Géolocalisation** et navigation intégrée
- **Système de favoris** et notes personnelles
- **Historique des services** par garage
- **Évaluations et recommandations**

### 🤝 **Location de Véhicules**
- **Contrats numériques** complets et légaux
- **Gestion des disponibilités** en temps réel
- **Calculs automatiques** de tarifs et cautions
- **Rapports d'état** détaillés avec photos
- **Suivi des revenus** et analytics de rentabilité
- **Communications intégrées** avec les locataires

### ☁️ **Synchronisation CloudKit**
- **Synchronisation automatique** entre tous vos appareils Apple
- **Sauvegarde sécurisée** dans le cloud
- **Partage familial** avec contrôle d'accès granulaire
- **Mode hors ligne** avec synchronisation différée

### 📸 **Gestion Avancée des Photos**
- **3 types d'images** : photo principale, photos supplémentaires, documents
- **Sélection multiple** depuis la galerie photos
- **Compression automatique** (800KB pour principale, 500KB pour autres)
- **Sauvegarde locale sécurisée** dans le dossier Documents
- **Chargement optimisé** avec mise en cache intelligente
- **Placeholders intelligents** générés selon la couleur et type du véhicule
- **Interface multilingue** (français/anglais) pour tous les éléments
- **Suppression sélective** avec confirmation utilisateur

### 🔐 **Sécurité & Authentification**
- **Apple Sign In** pour une connexion sécurisée et rapide
- **Chiffrement end-to-end** de toutes les données sensibles
- **Authentification biométrique** (Face ID / Touch ID)
- **Contrôle d'accès** par fonctionnalité

## 🚀 **Installation & Configuration**

### Prérequis
- Xcode 15.0+
- iOS 16.0+
- Compte développeur Apple (pour CloudKit)
- Swift 5.9+

### Installation
```bash
# Cloner le repository
git clone https://github.com/Andyg971/WheelTrack.git

# Naviguer dans le dossier
cd WheelTrack

# Ouvrir le projet dans Xcode
open WheelTrack.xcodeproj
```

### Configuration CloudKit
1. Créer un container CloudKit dans le portail développeur Apple
2. Configurer les schémas de données (automatique au premier lancement)
3. Activer les notifications push
4. Configurer les permissions de partage
